# 🚀 Self-Improving AI Agent

A modern web application that simulates an AI agent capable of **iterative self-improvement**.

The system generates multiple versions of responses, evaluates them, and improves them step-by-step to deliver better and more personalized outputs.

---

## 🔥 Features

- ⚡ Multi-version AI responses (V1 → V2 → V3)
- 🧠 Personalized answers based on user profile
- 📊 Progress tracking dashboard
- 🔄 Self-improvement feedback loop
- 🎨 Modern UI with animations

---

## 🛠 Tech Stack

- React (Vite)
- JavaScript
- CSS (Custom styling)
- Git & GitHub

## 📸 Screens
